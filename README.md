# Ambient Scribe — Indian Players

A landscape scan of ambient clinical documentation and AI medical scribe tools relevant to the Indian healthcare market.

---

## Contents

**`V2DD_Excel_Compendium.xlsx`**:

- **Landscape** — one row per product; columns cover pricing, India relevance, ambient vs. dictation classification, multi-speaker capability, Indian language support (English, Hindi/Hinglish, Bengali, Tamil, Kannada, Telugu, Malayalam), publicly found validation evidence, and due-diligence questions
- **Summary** — aggregate counts across the 16 mapped players
- **Sources** — all primary URLs with a note on what each was used to evidence
- **Read Me** — scope, language-coding conventions, and interpretation notes

---

## Players covered (16)

Augnito, MedoriaX, RxNote, EkaScribe, Merago AI Scribe, KiraDoc, HealthOrbit, QModify, CurePilot, VoxDox, DoctorsApp, Adrine, VivaLyn Labs, ScribeMedics, PenBOT, ScribeMed — plus Suki AI and Fireflies.ai as international benchmarks.

---

## Language coding convention

| Value | Meaning |
|-------|---------|
| Yes | Explicit public claim for that language |
| Likely | Broad multilingual claim without itemised list |
| Not public | No reliable public evidence found |

---

## Key caveats

- Desk-based scan using public websites and marketing pages only — not live demos or clinical pilots.
- Independent peer-reviewed validation for Indian-language ambient scribing was not found for any player; vendor accuracy claims are recorded separately.
- DPDPA compliance was explicitly claimed by only one player (Merago); others require direct vendor confirmation.
